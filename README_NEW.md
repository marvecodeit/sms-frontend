# SMS Frontend - School Management System

A modern, production-ready School Management System frontend built with Next.js 14, TypeScript, Tailwind CSS, and Shadcn UI.

## Features

✨ **Core Features**
- 🔐 Multi-role authentication (9 roles)
- 📊 Role-based dashboards
- 👥 Complete student & staff management
- 📈 Results & academic tracking
- 💬 Real-time chat & messaging
- 📱 Fully responsive design
- 🌓 Dark/light theme support
- ⚡ Type-safe with TypeScript

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn UI
- **State Management**: Zustand + React Query
- **Forms**: React Hook Form + Zod
- **Real-time**: Socket.IO
- **API Client**: Axios
- **Charts**: Recharts
- **Animations**: Framer Motion

## Installation

1. **Navigate and setup**
```bash
cd frontend
cp .env.example .env.local
```

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run type-check` - TypeScript check
